React

Na tomto dvoudenním školení se naučíte psát plnohodnotné aplikace v React JS. React je populární knihovna od Facebooku, ve které je již napsáno stovky tisíc aplikací a píší se další. Poznejte tuto ohromující technologii a začněte vytvářet aplikace, které dnes mění svět - aplikace v moderním Javascriptu a knihovně React. Školení probíhá na vlastních počítačích (na operačním systému nezáleží).
